For security reasons, the users.json file, which is used to set admin passwords is not tracked by the repository and needs to be manually set on the server. The JSON template is on the following line. DO NOT LEAVE THE _SAMPLE_ENTRIES_ AS THEY ARE.
[{"username":"_user_name_","password":"_password_","id":0},{"username":"_user_name_","password":"_password_","id":1}]
